# FrNourish

FrNourish is a simple and user-friendly nutrition and food awareness web application designed to help users make better food choices in an easy and accessible way. The project is built with a clean interface so that people of all age groups can use it comfortably.

## Features

- Simple and responsive user interface
- Easy navigation for all age groups
- Food and nutrition focused design
- Interactive frontend using JavaScript
- Clean styling with CSS
- Lightweight and easy to use

## Tech Stack

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- Google Cloud Run
